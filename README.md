# iso-time-parser

[ISO 8601](https://www.wikiwand.com/fr/ISO_8601) Time parser (not date, nor duration)

## Installation

```bash
npm i iso-time-parser
```

## Usage

```typescript
import { ... } from 'iso-time-parser';
```

## Development

**Install deps**

```bash
npm i
```

**Build/bundle package**

```bash
npm run build
```
